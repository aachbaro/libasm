## libasm (42)

Projet visant à implémenter en assembleur x86-64 un ensemble de fonctions standards de la libc.

### Contenu

- Implémentation de fonctions basiques en assembleur
- Respect des conventions d’appel
- Interaction avec le C pour les tests

### Contexte

Projet de l’école 42 ayant pour objectif de comprendre le fonctionnement bas niveau des appels de fonctions, de la mémoire et du lien entre C et assembleur.
